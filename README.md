# rx-block-unbrowser
一键导入 https://greenify.github.io/kr328/rx-block-unbrowser

用于屏蔽 非浏览器应用 的 默认关联

### 依赖
需要配合 [Riru-IFWEnhance](https://github.com/Kr328/Riru-IFWEnhance) 使用

  * IDM+

  * 网易云音乐极速版
  * 必收
